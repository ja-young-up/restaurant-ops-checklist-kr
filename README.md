# 외식업 매출·운영 체크리스트

국내 자료를 바탕으로 재구성한 외식업 자영업자용 반응형 체크리스트 페이지입니다. 법규 점검 중심이 아니라 실제 매출과 운영 효율을 높이는 실전 항목에 초점을 맞췄습니다.

## 포함 내용

- 신규 유입 만들기
- 플레이스·주문 전환 올리기
- 객단가 올리기
- 재방문·리뷰 관리
- 배달·포장 수익성
- 피크타임 운영 효율
- 원가·인건비·주간 관리

## 파일 구성

- `index.html`: 페이지 마크업
- `style.css`: 반응형 스타일
- `script.js`: 체크리스트 데이터, 검색, 진행률, 로컬 저장

## 참고

- 체크 상태는 브라우저 `localStorage`에 저장됩니다.
- 체크리스트 문장은 네이버 플레이스·예약주문, 소상공인시장진흥공단 상권정보, 기업마당 등 국내 자료를 바탕으로 실무 액션으로 재구성한 내용입니다.
