# 외식업 운영 체크리스트

국내 공공·기관 자료를 바탕으로 정리한 요식업 자영업자용 반응형 체크리스트 페이지입니다.

## 포함 내용

- 오픈 준비
- 위생·식품안전
- 인사·노무
- 세무·정산
- 매장·시설·안전
- 마케팅·고객관리
- 경영관리

## 파일 구성

- `index.html`: 페이지 마크업
- `style.css`: 반응형 스타일
- `script.js`: 체크리스트 데이터, 검색, 진행률, 로컬 저장

## 참고

- 체크 상태는 브라우저 `localStorage`에 저장됩니다.
- 법령·행정기준은 바뀔 수 있으므로 실제 적용 전 최신 공고와 관할 지자체 기준을 함께 확인해야 합니다.
